# EK301S_Prelim_Report
Preliminary Design Report for ENG EK 301S, where we made two preliminary truss designs and analyzed them with MATLAB code. We tried to predict load-cost ratio and maximum supportable load in a probabilistic framework using an empirically derived relationship for critical buckling load.

TrussJuly2.mlx contains the general truss analysis code. Each section is annotated and explained. 

Truss1_Setup.m and Truss2_Setup.m are MATLAB scripts to initialize/input our first and second preliminary truss designs, respectively. They are not .mat files since this allows for easy, quick changes to truss struture.  

createArchTruss.m, july7.py, and TrussOptimizationJuly7.m were used in the design of truss 2.
